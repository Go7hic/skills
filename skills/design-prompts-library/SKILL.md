# Design Prompts Library

## Overview

Use this skill when the user wants a style recommendation from the Design Prompts collection or asks to browse the available aesthetics before implementation.

## How to Use

- Pick the closest style skill from the list below.
- Load that style's own skill for implementation details.
- Preserve the user's product brief and adapt the style rather than copying the demo literally.
